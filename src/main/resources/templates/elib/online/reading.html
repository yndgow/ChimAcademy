시요일이란

출판사 창비의 시(詩) 큐레이션 전문 어플리케이션으로, 4만 3천여편 아름다운 시의 감동을 모두 담아 언제 어디서든 원하는 시를 찾아 읽고 선물할 수 있습니다. 부산광역시 전자도서관 회원이라면 무료로 모든 콘텐츠 이용이 가능합니다.


소개영상


주요기능

  • 오늘의 시 : 하루 한편씩 그날에 어울리는 시 선물(알림 Push)
  • 테마별 추천시 : 상황에 맞게 시요일이 엄선한 주제별 추천시
  • 시인 낭송 : 시인들의 음성을 직접 듣는 시
  • 키워드/태그 검색 : 본문·시어·시인·태그(주제어)를 이용한 빠르고 강력한 검색
  • 시요일의 선택 : 책∙영화∙반려동물 등 전문 필진의 유익한 글
  • 시作! 일기 : 당신의 시를 남겨보세요
  • 스크랩/SNS 공유 : 나만의 시집을 만들고 좋은 시는 예쁜 이미지로 공유

앱다운로드

구글 플레이 스토어 또는 앱스토어에서 '시요일' 을 검색하시면 어플리케이션을 다운로드 할 수 있습니다.

안드로이드 (Android)
구글플레이스토어 다운로드

IOS (Iphone)
앱스토어 다운로드


이용방법

  1. 앱 다운로드 후 설치
  2. 로그인 화면에서 '도서관 회원' 클릭 후 '부산도서관' 선택
  3. 부산광역시 통합회원 아이디로 로그인
  4. 자유롭게 시요일 홈페이지 이용하기
  5. ※ 준회원(웹회원)은 가입할 수 없으며, 도서관을 방문하여 회원증을 발급받은 후 이용하여야 합니다.

시요일 도서관 홈페이지 이용방법

  1. 부산광역시 전자도서관 홈페이지 로그인
  2. 상단 온라인콘텐츠 메뉴 내 '시요일' 클릭
  3. 자유롭게 시요일 홈페이지 이용하기
